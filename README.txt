If you have been problem to Ubuntu OS find the wireless or wifi drive, so this repository can help you.

This repository you can find some files:

- README.txt
- iwlwifi-9000-pu-b0-jf-b0-34.618819.0.tgz (drive wireless or wifi in compacted form);
- install_wifi_dispositive.sh (file with script commands to export drive's files, install drive, remove the files exporteds and shutdown the OS);

Atention: I'm using Ubuntu OS version 22.04 LTS. Maybe, if you are using other Ubuntu version, you can use this repository too.

If you want to execution the install_wifi_dispositive.sh, you just need run this command:

bash install_wifi_dispositive.sh
